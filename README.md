# Chatbot for Healthy Ageing
##  Project Description
The purpose of this project is to develop a conversational AI/chatbot for older adults. The goal of the chatbot is to help older adults maintain good health, focusing on nutrition. The functions of the chatbot are to be accessible, personalized, and reliable. The chatbot will use OpenAI’s language model as its core technology.



## Features

- Health and Nutricnal guidance 
- Live responses
- Powered by OpenAI’s Language Model
- Designed for Older Adults
